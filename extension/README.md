# WA Chat Summariser Extension Starter

A minimal Chrome extension starter template built for the `extension/` folder.

## Files included

- `manifest.json` - Chrome Extension Manifest V3
- `background.js` - background service worker
- `popup.html` - popup UI
- `popup.js` - popup behavior
- `styles.css` - basic popup styling
- `icons/` - starter SVG icon assets

## How to load in Chrome

1. Open `chrome://extensions` in Chrome.
2. Enable `Developer mode`.
3. Click `Load unpacked`.
4. Select the `extension/` folder.

## Notes

- This starter includes a simple popup UI and a message listener in the background worker.
- Replace the sample summarisation logic in `popup.js` with your own WhatsApp chat processing code.
